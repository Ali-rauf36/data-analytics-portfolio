# E-Commerce Analytics Pipeline

**Tools:** SQL (SQLite) · Python (pandas) · Power BI  
**Dataset:** UK Online Retail · 541,909 transactions · Dec 2010 – Dec 2011

## Pipeline Architecture
Raw CSV → SQLite Database → Python Analysis → Power BI Dashboard

## Key Findings
- November 2011 peak revenue £1.5M — pre-Christmas holiday surge
- UK dominates revenue (~90%); Netherlands leads international sales
- REGENCY CAKESTAND 3 TIER and PAPER CRAFT top products by revenue
- 79.9% of customers are one-time or occasional buyers — retention problem
- Regular + Loyal customers (~20%) drive majority of revenue — Pareto distribution

## Skills Demonstrated
- End-to-end data pipeline: SQL → Python → Power BI
- SQLite database creation and querying from Python using sqlite3
- Customer segmentation with pandas cut() and groupby()
- Time series analysis and seasonal trend identification
- Navy & Gold themed Power BI dashboard with DAX measures
