# COVID-19 Global Data Explorer

**Tools:** Python (pandas, matplotlib, seaborn) · Jupyter Notebook  
**Dataset:** Our World in Data — 243 countries, Jan 2020 – Oct 2023

## Key Findings
- US leads in total cases; Brazil and Russia have disproportionately high deaths
- Omicron wave (2022) caused the largest single case spike of the pandemic
- Smaller nations achieved higher vaccination rates than large ones
- Africa, North America and South America exceeded 1% case fatality rate

## Skills Demonstrated
- Working with large, messy real-world datasets (350K+ rows, 67 columns)
- Time series analysis and trend identification
- Dual axis charts, conditional coloring, reference lines
- Population-adjusted rate calculations
